== Description ==

This plugin adds the shortcode [coolEye] which embeds a CoolIris wall into your post or page.
Many different built in search modes for YouTube, Flickr, and Picasa, including keywords,
username, channel, album, favorites, and playlists. Also supports any Media RSS feed.
Adds a new button to the Wordpress TinyMCE editor which brings up a modal dialog box
for easily creating shortcodes.

Many Media RSS providers can be found here: http://www.cooliris.com/channels/#providers

== Installation ==

1. Use the WordPress Plugin Installer... or...
2. Upload `coolEye` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Adjust the default parameters for the shortcode in the WordPress Settings->CoolEye area
5. Have fun inserting shortcodes

== Frequently Asked Questions ==

= What about some of the other CoolIris features like Facebook, etc... =

When CoolIris releases these features, I'll try to update this plugin ASAP

= There's some other parameters supported by CoolIris that aren't in the plugin =

Let me know which ones you need, and I'll try to put them in the next update

= What's the deal with CoolIris's license? =

Short summary - if your site is not for profit, the CoolIris embed is free. YMMV, Offer not valid where prohibited.

Check out: http://www.cooliris.com/legal/terms/

= What other software does CoolEye require from user browsers =

CoolIris is Flash based. CoolEye will only work with Flash enabled browsers.
